# 商品批量删除 - 快速参考

## 🎯 功能入口

**位置**: `/products` 页面 → 统计信息栏 → 批量操作按钮

## 🔘 操作按钮

| 按钮 | 功能 | 颜色 | 影响范围 |
|------|------|------|----------|
| 🗑️ 删除不推荐 | 删除推荐分数 < 50 的商品 | 橙色 | 部分商品 |
| 🗑️ 清空列表 | 删除所有商品 | 红色 | 全部商品 |

## ⚙️ 推荐分数阈值

- **默认值**: 50 分
- **含义**: 低于此分数的商品被视为"不推荐"
- **判断标准**:
  - < 30: 不推荐（市场潜力低）
  - 30-50: 谨慎考虑（竞争激烈）
  - 50-70: 可以考虑（有一定潜力）
  - > 70: 推荐（高潜力商品）

## 📋 操作流程

```
1. 点击删除按钮
   ↓
2. 查看确认对话框
   ↓
3. 确认或取消
   ↓
4. 等待删除完成
   ↓
5. 查看结果通知
   ↓
6. 列表自动刷新
```

## ⚠️ 注意事项

- ❌ **不可恢复** - 删除后无法撤销
- 🔗 **级联删除** - 相关趋势历史记录也会被删除
- 💾 **建议备份** - 重要数据删除前先备份
- 🧪 **先测试** - 生产环境操作前先在测试环境验证

## 🔌 API 端点

```
DELETE /api/trends/products/bulk-delete
```

**请求示例**:
```json
{
  "action": "delete_not_recommended",
  "threshold": 50
}
```

## 📊 使用场景

### 删除不推荐商品
- ✅ 定期清理低质量商品
- ✅ 优化商品数据库
- ✅ 保持商品列表精简
- ✅ 提高数据质量

### 清空商品列表
- ✅ 重新开始采集
- ✅ 切换业务方向
- ✅ 测试环境清理
- ✅ 系统维护重置

## 🚀 快捷键

目前不支持快捷键，建议未来版本添加：
- `Ctrl + Shift + D` - 删除不推荐
- `Ctrl + Shift + X` - 清空列表（需二次确认）

## 📈 最佳实践

1. **定期清理** - 每周清理一次不推荐商品
2. **数据备份** - 清空前导出重要数据
3. **分批操作** - 大量数据时分批删除
4. **监控日志** - 记录删除操作和数量
